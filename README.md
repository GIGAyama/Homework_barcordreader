# 🎒 宿題ポスト Enterprise Edition

「宿題ポスト」は、小学校の教室に欠かせない **宿題の提出確認** と **朝の健康観察（きもちメーター）** を、ChromebookやiPadから一瞬で完了できるWebアプリケーションです。

商用SaaSレベルの美しいUI/UXと、Google Apps Script (GAS) 特有の通信の不安定さをカバーする「Auto-Retry通信エンジン」を搭載し、教育現場で **「絶対に落ちない、迷わない」** 最高の体験を提供します。

## ✨ 主な特徴 (Features)

*   **🚀 ゼロ・コンフィグ（完全自動セットアップ）**
    
    *   コードを貼り付けて開くだけ！面倒なスプレッドシートの準備はすべて裏側で自動構築されます。
        
*   **💳 バーコード ＆ テンキー入力**
    
    *   低学年の児童でも、自分のバーコード（ID）をかざすか、数字を入れるだけで直感的に操作できます。
        
*   **❤️ 朝の「きもち」メーター搭載**
    
    *   宿題提出と同時に「げんき😊」「ねむい🥱」などを選択。先生はダッシュボードで「今日フォローが必要な子」を朝の会前に一目で把握できます。
        
*   **📊 先生用ダッシュボード ＆ 分析機能**
    
    *   今日の提出率、きもちの割合、未提出者のリストアップから、日々の課題ルールの柔軟な設定まで、すべてスマホやタブレットから管理可能です。
        
*   **👧 低学年対応（全漢字ルビ付き）**
    
    *   児童が触れる画面のすべての漢字にふりがなが振ってあります。
        
*   **✉️ 未提出者の自動メール通知**
    
    *   毎日指定した時間に、宿題を出していない児童のリストを先生に自動送信します。

## 🛠️ 導入方法 (Installation)

プログラミングの知識は一切不要です。以下の手順で5分で導入できます。

### 1. スプレッドシートの準備

1.  Googleドライブを開き、**新しいスプレッドシート** を作成します。
    
2.  上部のメニューから `[拡張機能]` > `[Apps Script]` をクリックします。
    

### 2. コードの貼り付け

1.  開いたApps Scriptの画面で、最初からある `コード.gs` の中身をすべて消し、このリポジトリの `code.gs` のコードを貼り付けます。
    
2.  左側の `[＋]` ボタンを押して `[HTML]` を選び、ファイル名を `index` にします。（`.html`は自動で付きます）
    
3.  できた `index.html` の中身をすべて消し、このリポジトリの `index.html` のコードを貼り付けます。
    
4.  💾 保存ボタン（または `Ctrl+S` / `Cmd+S`）を押します。
    

### 3. デプロイ（アプリとして公開）

1.  画面右上の青い **`[デプロイ]`** ボタンを押し、**`[新しいデプロイ]`** を選びます。
    
2.  左側の歯車マーク ⚙️ を押し、**`[ウェブアプリ]`** を選びます。
    
3.  以下の設定にします。（**※超重要！**）
    
    *   次のユーザーとして実行: **自分**
        
    *   アクセスできるユーザー: **全員**
        
4.  **`[デプロイ]`** ボタンを押します。
    
5.  *(初回のみ)* 「アクセスを承認」画面が出ます。「詳細を表示」→「（プロジェクト名）に移動（安全ではないページ）」をクリックして許可してください。
    

### 4. アプリの起動

デプロイ完了後に表示される **ウェブアプリのURL** をブラウザで開いてください。 初回アクセス時に「魔法のじゅんびをしています...」と表示され、自動的に必要なシートがスプレッドシートに作成されます！

## 📖 使い方 (Usage)

### 👨‍🏫 先生の設定（管理者メニュー）

1.  アプリ右上の **歯車アイコン ⚙️** をクリックします。
    
2.  デフォルトのPINコード **`1234`** を入力してログインします。
    
3.  `[設定]` タブから、PINコードや通知時間を変更してください。
    
4.  `[名簿]` タブから、児童のIDと名前を登録してください。
    
5.  `[課題]` タブから、提出させたい宿題のルール（毎日、曜日固定など）を設定します。
    

### 👦 児童の操作

1.  教室の端末でアプリを開いておきます。
    
2.  バーコードリーダーで自分のIDを読み取るか、テンキーでIDを入力します。
    
3.  自分の名前が表示されたら、今日出す宿題をタップして `[つぎへ]` を押します。
    
4.  今日の「きもち」をタップすると提出完了し、紙吹雪が舞います！🎉
    

## 💻 技術スタック (Tech Stack)

*   **Backend:** Google Apps Script (Container-bound Script)
    
*   **Frontend:** React 18, Babel (Standalone)
    
*   **Styling:** Tailwind CSS (CDN)
    
*   **Animations:** Canvas Confetti
